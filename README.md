# TechFlow Solutions Website

A modern, responsive business website for TechFlow Solutions - a technology consulting company.

## Features

- ✨ Modern responsive design
- 🎯 Hero section with compelling call-to-action
- 📋 About Us section with company information
- 🛠️ Services section with feature cards
- 💬 Customer testimonials section
- 📞 Contact form for inquiries
- 📱 Mobile-responsive navigation
- 🎨 Professional blue & white color scheme
- ⚡ Smooth scrolling animations
- 🚀 Fast loading and optimized code

## Technologies Used

- HTML5 for semantic structure
- CSS3 for styling and animations
- Vanilla JavaScript for interactivity
- Responsive design principles
- Mobile-first approach

## Deployment

This website is deployed using GitHub Pages and can be viewed at: 
[Live Website](https://unitrecords2021-hub.github.io/techflow-website/)

## Local Development

Simply open `index.html` in your web browser to view the website locally.
